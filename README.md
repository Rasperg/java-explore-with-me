# java-explore-with-me
Приложение позволяет пользователям делиться информацией об интересных событиях и находить компанию для участия в них.

Стек:
Java 11, 
Spring Boot, 
Hibernate, 
PostgreSQL, 
Maven, 
Lombok, 
Postman, 
Docker.
